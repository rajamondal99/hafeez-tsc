Print Customer Receipt and Send by Email

This module will allows to print report for
  * Customer Payment Receipt


And while validating customer payment it will send email to customer with attachment receipt pdf report.

Configuration: You will find checkbox on customer form called "Send Receipt By Mail" if this box is ticked then only it will send email to customer while validating payment.

Main features are:
  * Allow user to print pdf report from payment form.
  * Allow user to setup configuration on customer form to send email with report while validating payment.
  * Template is available under settings/.../Templates
