Print Payment Receipt - by Invoice

This module allow user to send receipt from invoice form. Thus module will consider all payment made against invoice and allow user to print and send receipt by email to customer.

Features List:
1. Print Payment Receipt from Invoice form.
2. Send Payment Receipt by email from Invoice form.

This module also send automatic send email to your customer if invoice is getting paid fully with payment receipt print copy. (Means when invoice state change to Paid.). This is based on configuration on customer form.

Configuration on Customer Form

If this checkbox is ticked then customer will be automatically notify by email with receipt printed copy when invoice get fully paid.
